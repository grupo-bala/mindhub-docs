# Aplicativo

Para as tecnologias do aplicativo Android escolhemos as seguintes tecnologias:

- [Kotlin](https://kotlinlang.org/): devido a sua recomendação pela Google
- [Android jetpack](https://developer.android.com/jetpack): por facilitar o desenvolvimento

Ademais, utilizaremos o [Android Studio](https://developer.android.com/studio) como ambiente de desenvolvimento.

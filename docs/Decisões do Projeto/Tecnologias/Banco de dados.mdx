# Banco de dados

 Para a tecnologia do banco de dados tinhamos algumas opções, sendo elas:

- [Postgresql](https://www.postgresql.org/)
- [MongoDB](https://www.mongodb.com/)
- [MySQL](https://www.mysql.com/)

Optamos por escolher o [Postgresql](https://www.postgresql.org/) como tecnologia para o banco de dados devido a nossa familiaridade com ele e por implementar um modelo relacional no banco de dados.

# API

Para as tecnologias do servidor escolhemos as seguintes tecnologias:

- [TypeScript](https://www.typescriptlang.org/): devido a tipagem estática que facilitará a produção de código ao longo do projeto
- [NestJS](https://nestjs.com/): pela facilidade de desenvolver usando o framework e pela integração com o swagger
- [TypeORM](https://typeorm.io/): integração com TypeScript e Postgresql
- [Vitest](https://vitest.dev/): devido a sua alta performance e integração com o TypeScript
- [Eslint](https://eslint.org/): consistência de estilo e boas práticas
